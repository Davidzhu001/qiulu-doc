# 其他方面

##  场地租借

1.添加场地:【场地/教室租借】&gt; 添加场地  

![](.gitbook/assets/2018-11-12_152417.png)

2.场地展示:【场地/教室租借】&gt; 场地列表  

![](.gitbook/assets/2018-11-12_152535.png)

{% hint style="info" %}
这个页面可以看到您预约的时间,预约的场地总个数以及一个场地的房间数
{% endhint %}

3.场地添加房间

方法一: 【场地/教室租借】&gt; 场地列表 &gt; 选取指定的场地,点击"操作"按钮

![](.gitbook/assets/2018-11-12_153208.png)

![](.gitbook/assets/2018-11-12_153035.png)

方法二: 【场地/教室租借】&gt; 场地列表 &gt; 选取指定的场地,点击"操作"按钮

![](.gitbook/assets/2018-11-12_153208.png)

![](.gitbook/assets/2018-11-12_153414.png)

4.添加房间的时间以及指定的人 :【场地/教室租借】&gt; 场地列表 &gt; 选取指定的场地,点击"操作"按钮

![](.gitbook/assets/2018-11-12_153208.png)

![](.gitbook/assets/2018-11-12_153639.png)

在页面的表格中,选取指定的房间名,相对应后面的时间,然后双击,弹出个框

![](.gitbook/assets/2018-11-12_153742.png)

## 标签管理

1.添加标签:【用户标签/分类】  

![](.gitbook/assets/2018-11-12_154452.png)

{% hint style="info" %}
点击"添加新标签"按钮,进行添加,若想要修改标签,点击标签旁边的"编辑"按钮
{% endhint %}

2.使用标签

方法一:在添加老师或学生信息时,选择指定的标签进行添加

方法二:【用户标签/分类】&gt; 选择指定的人后点击"查看"按钮 &gt; 编辑信息 选择指定的标签,点击提交即可

![](.gitbook/assets/2018-11-12_154931.png)

![](.gitbook/assets/2018-11-12_155012.png)

{% hint style="info" %}
这主要针对于个别需要添加的
{% endhint %}

方法三:【学生管理】&gt; 所有学生

![](.gitbook/assets/2018-11-12_155435.png)

{% hint style="info" %}
点击全选按钮,会出现"批量添加标签"按钮,点击进行批量添加
{% endhint %}

## 短信通知

1.新建群发模板:【短信群发】&gt; 新建短信群发模板  

![](.gitbook/assets/2018-11-12_155806.png)

{% hint style="info" %}
请严格按照上面的温馨提示,不然短信模板可能通不过
{% endhint %}

2.新建自定义模板:【短信群发】&gt; 新建短信

![](.gitbook/assets/2018-11-12_160343.png)

{% hint style="info" %}
可以选择发送老师还是学生
{% endhint %}

3.短信发送记录:【短信群发】&gt;群发记录

![](.gitbook/assets/2018-11-12_160533.png)

4.短信使用情况:【短信群发】&gt; 短信报告

![](.gitbook/assets/2018-11-12_160654.png)

{% hint style="info" %}
上面记录短信使用记录以及短信使用条数和剩余条数
{% endhint %}
